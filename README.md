# Jacks Pipe and Civil - Website

A professional one-page website for Jacks Pipe and Civil Pty Ltd, a civil construction company based in South East Queensland, Australia.

## About

Jacks Pipe and Civil provides quality civil pipe laying services including:
- Pipeline Construction
- Civil Construction
- Manhole Construction
- Live Connections
- Backfilling & Road Crossings

## Deployment

### Vercel (Recommended)

1. Push this repository to GitHub
2. Connect your GitHub repository to Vercel
3. Vercel will automatically detect the static site and deploy

Or deploy directly:
```bash
npx vercel
```

### Other Static Hosts

This is a simple static HTML site that can be deployed to any static hosting service (Netlify, GitHub Pages, etc.)
